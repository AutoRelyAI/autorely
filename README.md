# AutoRely

AI-powered multilingual changelogs from your GitHub commits.

One push. Every audience. Every language.

## What is AutoRely?

AutoRely connects to your GitHub repo and automatically generates polished changelogs from your commit history — for developers, customers, and PMs — in multiple languages.

## Why?

- Writing changelogs manually is tedious
- Commit messages are for devs, not end users
- Global products need multilingual release notes

AutoRely solves all three with one click.
